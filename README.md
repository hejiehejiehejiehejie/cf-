# QwQ - Codeforces 沉浸式阅读与深色引擎

为 Codeforces 题目页面设计的轻量级用户脚本，致力于提供最纯粹的阅读体验与现代化的深浅色主题，完美兼容各类第三方美化插件。

[点击安装](https://github.com/hejiehejiehejiehejie/cf-/raw/refs/heads/main/QwQ.user.js)

## ✨ 核心功能

* **📖 沉浸式阅读视图**：一键隐藏 Codeforces 侧边栏，正文及提交通道居中对齐，并限制最大阅读宽度（`1100px`），提供最符合人体工学的代码阅读体验。
* **🌙 原生深色引擎**：自带轻量级全局深色模式，脱离对外部插件的依赖。**独家智能反色 Codeforces Logo**（去除刺眼白底），并为顶部导航栏引入现代化的**毛玻璃 (Glassmorphism)** 效果。
* **👻 幽灵悬浮窗 (Ghost FAB)**：极致极简设计。操作按钮以极低的透明度（15%）悬浮于屏幕右下角，平时几乎隐形，不打扰视线，鼠标靠近时平滑浮现。
* **⚙️ 状态无缝记忆**：利用 `localStorage` 自动保存你的“深浅主题”与“阅读视图”偏好，刷新或打开新页面依旧保持最佳状态。
* **🤝 极高兼容性**：深度重构底层 CSS 逻辑，彻底释放页面全局尺寸劫持，与 **CodeforcesBetter** 等扩展 100% 完美兼容，拒绝面板错位。

## 🚀 快捷使用

本脚本支持无缝的快捷键操作与鼠标交互：

* **切换深色模式**：按下 `Alt + D` 或点击右下角 🌙/☀️ 悬浮图标。
* **切换阅读视图**：按下 `Alt + Q` 或点击右下角 📖 悬浮图标隐藏/显示侧边栏。
* *注：在代码编辑器或输入框内打字时，快捷键会自动屏蔽以防止误触。*

## ❓ 常见问题

* **脚本未生效？** 确保你的页面处于题目详情页，匹配 `@match` 规则（如 `https://codeforces.com/problemset/problem/*/*`）。
* **页面有横向滚动条？** 脚本已强制裁切整页横向滚动。对于超宽的表格或长代码块，已开启**局部滚动**，鼠标移入对应区域滑动即可。
* **和 CodeforcesBetter 冲突吗？** 完全兼容！建议**关闭** CodeforcesBetter 自带的深色模式，使用本脚本的原生深色引擎以获得更好的 Logo 与导航栏视觉体验。
