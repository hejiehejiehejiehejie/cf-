# QwQ - Codeforces 阅读优化脚本

一个用于 Codeforces 题目页面的轻量级用户脚本。主要提供侧边栏隐藏、原生深色模式以及界面布局调整功能，以改善代码与题目的阅读体验。

[点击安装](https://github.com/hejiehejiehejiehejie/cf-/raw/refs/heads/main/QwQ.user.js)

## 核心功能

* **阅读视图**：隐藏右侧边栏，将题目正文与提交通道居中显示。限制了最大阅读宽度（`1100px`），并自适应左右留白，避免宽屏下文字过长影响阅读。
* **深色模式**：脚本内置了基础的深色样式引擎。开启后，会对网页背景、字体、代码块和表格进行深色渲染。同时对顶部 Codeforces Logo 进行了反色处理，并对导航栏应用了半透明效果，以适配深色背景。
* **右下角悬浮窗**：将操作按钮统一放置在页面右下角。默认状态下按钮透明度降低至 15%，鼠标悬停时恢复正常，以减少对正文的视觉干扰。
* **状态记忆**：利用浏览器的 `localStorage` 记录用户的深色模式与阅读视图开关状态，刷新页面后自动应用。
* **扩展兼容**：移除了对全局 `html` 与 `body` 的强制宽度限制，改为局部对齐，目前能够正常兼容 CodeforcesBetter 等改变页面布局的第三方插件。

## 快捷操作

支持通过鼠标点击右下角悬浮图标，或使用以下快捷键进行操作（输入框内打字时会自动屏蔽以防误触）：

* `Alt + D`：切换深色 / 浅色模式
* `Alt + Q`：切换阅读视图（显示 / 隐藏侧边栏）

## 常见问题

* **页面出现横向滚动条？** 脚本默认裁切了整页的横向溢出。对于过宽的表格或长代码块，已开启局部滚动，鼠标移入该区域滑动即可。
* **与 CodeforcesBetter 同时使用？** 两者可同时运行。为避免样式冲突，建议关闭 CodeforcesBetter 自带的深色模式，使用本脚本提供的深色主题即可保证界面（如 Logo 和导航栏）的正常显示。
* **脚本未生效？** 请确认当前所在页面属于题库或比赛题目详情页（URL 匹配 `https://codeforces.com/problemset/problem/*/*` 等规则）。
